Core Features

Built for SunSDR2 DX

Mobile First

Touch-optimized UI with safe-area adaptation, PWA manifest, and service worker. HTTPS/WSS secure context required for iOS Safari microphone and AudioContext.

Implemented

Real-Time Waterfall

512-bin FFT spectrum at ~38 Hz. Adaptive per-row noise floor (30th percentile), blue-sea baseline bias, contrast gain stretching, and black→blue→cyan→yellow→red colour ramp.

Implemented

Dual-Codec RX Audio

Tagged dual-codec transport: 0x00 = Int16 PCM (~256 kbps), 0x01 = Opus 16 kHz mono (~18-24 kbps, >10× bandwidth savings). Runtime-switchable via Audio Codec menu. Falls back to PCM if libopus unavailable.

Implemented

TX Voice Modulation

Browser microphone → AudioWorklet 48k→16k downsample → Opus/PCM tagged uplink → fractional resampler → overlap-save Hilbert SSB (USB/LSB) → 24-bit IQ → 0xFFFD TX stream. On-air verified: Tune ~12 W, SSB voice 30–40 W PEP.

Implemented

Per-Band TX Power

Device DRIVE command (0x0017) with square-root taper. Drive byte in trailing word. Per-band power panel → /api/band_power → band_power.json. Re-sent on every QSY and PTT assert.

Implemented

TX Telemetry (0x1F00)

Device sends telemetry in all modes (RX/TX/TUNE). off14 u16 = pwr_raw, off16 u16/100 = SWR (1.32–1.37 verified), off18 f32 = temp °C. Power via cubic fit: (pwr_raw-9)³ × 1.91e-5 watts.

Implemented

PTT Safety System

ACK retry with 3× redundancy, backup s: force-RX command, frontend watchdog timer, backend forced-RX handler on stop. Touch-move auto-release on mobile.

Implemented

WDSP Noise Reduction

Optional libwdsp.dylib integration: NR2 spectral noise reduction, AGC (4 modes), ANF auto-notch, NB noise blanker. Runtime-toggled — RX works without it. UI queries availability via getWDSPStatus:.

Implemented (optional)

Sample Rate Selector

IQ bandwidth: 39 / 78 / 156 / 312 kHz. Set via 0x0001 HW_INIT word[11] = rate index. Full re-boot sequence on rate change. Frontend menu + backend setSampleRate:.

Implemented

ATT / Preamp Control

Hardware attenuator: 0=-20dB, 1=-10dB, 2=0dB, 3=+10dB. Memory channels with JSON persistence. AGC, filter, and mode controls via /WSCTRX command protocol.

Implemented

Quick Start

Deploy in 5 Minutes

# 1. Clone & install dependencies git clone https://github.com/cheenle/sunsdr.git cd sunmrrc ../venv/bin/pip install fastapi uvicorn numpy scipy # 2. Place TLS certs (or use DISABLE_SSL=1 for HTTP dev) # certs/fullchain.pem + certs/radio.vlsc.net.key # 3. Start ./restart.sh # background, https://localhost:8889 WEB_PORT=8080 ./restart.sh # custom port ./restart.sh -f # foreground (live logs) # 4. Open in browser https://radio.vlsc.net:8889
