核心功能

为 SunSDR2 DX 打造

移动优先

触控优化界面，支持安全区适配、PWA 安装到桌面。HTTPS/WSS 确保 iOS Safari 麦克风和 AudioContext 正常工作。

已实现

实时频谱瀑布

512 点 FFT 频谱 ~38 Hz 刷新。自适应底噪（30% 百分位）、蓝色基底偏移、 对比度拉伸、黑→蓝→青→黄→红色带渲染。S 表指数平滑。

已实现

双编解码 RX 音频

标签式双编解码传输：0x00 = Int16 PCM (~256 kbps)、 0x01 = Opus 16kHz 单声道 (~18-24 kbps，>10× 省流)。 Audio Codec 菜单实时切换，libopus 不可用时自动回退 PCM。

已实现

TX 语音发射

浏览器麦克风 → AudioWorklet 48k→16k 降采样 → Opus/PCM 标签上行 → 连续分数重采样 → 重叠保留 Hilbert SSB (USB/LSB) → 24-bit IQ → 0xFFFD TX 流。 实测：Tune ~12W，SSB 语音 30–40W PEP。

已实现

分波段功率控制

设备 DRIVE 命令 (0x0017)，平方根锥度。 Drive 字节放 trailing word。波段功率面板 → /api/band_power → band_power.json。每次 QSY / PTT 重发。

已实现

TX 遥测 (0x1F00)

设备在所有模式（RX/TX/TUNE）下发送遥测。off14 u16 = 功率原始值、 off16 u16/100 = 驻波比（已验证 1.32–1.37）、off18 f32 = 温度 °C。 功率三次方拟合：(pwr_raw-9)³ × 1.91e-5 W。

已实现

PTT 安全保护

ACK 确认 + 3 次重试，备用 s: 强制 RX 命令， 前端 watchdog 定时器，后端强制 RX 处理。手机端触控滑动自动释放。

已实现

WDSP 专业降噪

可选 libwdsp.dylib 集成：NR2 频谱降噪、AGC（4 种模式）、ANF 自动陷波、 NB 噪声消除。运行时切换 — 无 WDSP 也能正常收听。UI 通过 getWDSPStatus: 查询可用性。

已实现（可选）

采样率选择器

IQ 带宽：39 / 78 / 156 / 312 kHz。通过 0x0001 HW_INIT word[11] = 速率索引设置。 切换采样率会触发完整重启动。前端菜单 + 后端 setSampleRate:。

已实现

ATT / 前置放大

硬件衰减器：0=-20dB, 1=-10dB, 2=0dB, 3=+10dB。 频率记忆频道 JSON 持久化。AGC、滤波器、模式通过 /WSCTRX 命令协议控制。

已实现

快速部署

5 分钟上线

# 1. 克隆仓库并安装依赖 git clone https://github.com/cheenle/sunsdr.git cd sunmrrc ../venv/bin/pip install fastapi uvicorn numpy scipy # 2. 放置 TLS 证书（或设置 DISABLE_SSL=1 用 HTTP 调试） # certs/fullchain.pem + certs/radio.vlsc.net.key # 3. 启动 ./restart.sh # 后台运行, https://localhost:8889 WEB_PORT=8080 ./restart.sh # 自定义端口 ./restart.sh -f # 前台运行（实时日志） # 4. 浏览器打开 https://radio.vlsc.net:8889

准备好开始了吗？
